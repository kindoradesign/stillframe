# Stillframe

Official downloads for **Stillframe**, a Windows app that saves clean full-page website captures at desktop, tablet, and mobile widths in one run.

## Free Beta v0.5.0-beta

- [Download for Windows](https://github.com/kindoradesign/stillframe/releases/download/v0.5.0-beta/Stillframe-Beta-Setup.exe)
- [Release notes and ZIP package](https://github.com/kindoradesign/stillframe/releases/tag/v0.5.0-beta)
- SHA-256: `7116c9b67529f54180f33a76fe66fe03b86b2dcaa7efa1f43441c5fe3817f653`

The free beta has no time limit and requires no card. Captures and entered URLs are processed locally and are not sent to Stillframe's operator.

### Main features

- Capture 1440 px, 768 px, and 390 px widths together
- Add custom widths
- Stabilize animations, transitions, and sliders
- Save full-page PNG or JPG images
- Capture localhost and local HTML
- Japanese and English interface

### Windows notice

This beta is not yet code-signed, so Windows may show an unverified-publisher warning. Download only from this repository and verify the SHA-256 checksum before running the installer. If you are uncertain, do not run it.
